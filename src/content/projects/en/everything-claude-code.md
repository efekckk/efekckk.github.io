Coding agents are only as good as the harness around them. This repo collects what
actually moved the needle for me: skill definitions, instinct-style guardrails,
persistent memory patterns and a research-first development loop.

## Why it exists

Most agent advice is vibes. This is the opposite — every pattern in the repo earned
its place by surviving daily production use across Claude Code, Codex, Opencode and
Cursor. If it didn't make the agent measurably better, it didn't get committed.
